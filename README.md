# 🎬 Vizard Pro AI

> AI-powered video repurposing tool for YouTube Shorts, TikTok and Facebook Reels.

Vizard Pro AI is an open-source desktop application that automatically transforms long videos into high-quality short clips using AI.

Unlike simple video splitters, Vizard Pro AI understands the content of the video, finds the most engaging moments, automatically reframes to 9:16, adds subtitles, and generates titles, descriptions, keywords and thumbnails.

---

# Features

## AI Analysis

- Faster-Whisper transcription
- Gemini Smart Analysis
- Viral moment detection
- Automatic clip scoring
- Smart hooks detection

---

## Smart Editing

- Automatic 9:16 reframing
- Face tracking
- Auto zoom
- Silence removal
